# humanlang
The source code of the humanlang module for machine learning low ressources support
# special dict structure
functions
```json
{
  "word": 
    "$$$set": "$$$func",
    "$$$func": my_function,
    "$$$argument": ['myfirstarg', 'mysecondarg']
}
```
default value
```json
{
  "word": {
    "$$$set": "hello",
    "hi": {
            "$$$set": "hi"
          }
  }
}
```
